Object detection

Object detection is a computer technology related to computer vision and image processing that deals with detecting instances of semantic objects of a certain class (such as humans, buildings, or cars) in digital images and videos.

This is an example of multiple object detection using pre trained model in TensorFlow.js

You can hold some objects up close to your webcam and it will be classified automatically and you will see their name inside a white box for each object that is being classified.
